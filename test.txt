The file names my_file, myfile2 and myfile3 are used for demonstration purposes.
They signify three different files and not the different versions of the same file.

You wish to add an existing file GitInPractice.asciidoc to the index staging area for
inclusion in the next commit.

Git doesn’t add anything to the index without your instruction. As a result, the first
thing you have to do with a file you want to include in a Git repository is request that
Git add it to the index.
